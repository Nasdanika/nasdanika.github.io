help 

Usage: nsd help [-ahHV] [-l=<level>] [-o=<output>] [COMMAND]
Outputs usage for all registred commands
  -a, --action-model      Output to action model
  -h, --help              Show this help message and exit.
  -H, --html              Output to HTML
  -l, --header-level=<level>
                          Starting level for HTML header tags in HTML output,
                            the default value is 1.
  -o, --output=<output>   Output file
  -V, --version           Print version information and exit.
Commands:
  site  Generates help HTML site
