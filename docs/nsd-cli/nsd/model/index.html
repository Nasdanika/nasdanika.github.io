model

Version: org.nasdanika.emf@2025.4.0
Usage: nsd model [-fhV] [--content-type-resource-factory=<String=Class>]... [--extension-resource-factory=<String=Class>]... [--protocol-resource-factory=<String=Class>]... <uri> [COMMAND] Loads EObject from a URI or file <uri> EObject URI or file path, resolved relative to the current directory --content-type-resource-factory=<String=Class> Maps content type to resource factory class --extension-resource-factory=<String=Class> Maps extension to resource factory class -f, --file URI parameter is a file path -h, --help Show this help message and exit. --protocol-resource-factory=<String=Class> Maps protocol to resource factory class -V, --version Print version information and exit. Commands:
  • crew-ai - Generates CrewAI code and configuration from a model
  • html-app - Generates html application model from a model
  • save - Saves model to a file
