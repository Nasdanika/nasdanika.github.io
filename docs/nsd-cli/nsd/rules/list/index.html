list

Usage: nsd rules list [-hrV] [--progress-console] [--progress-data] [--progress-json] [-o=<output>] [--progress-output=<progressOutput>] [--exclude-rule [=<ruleExcludes>...]]... [--exclude-rule-set [=<ruleSetExcludes>...]]... [--include-rule [=<ruleIncludes>...]]... [--include-rule-set [=<ruleSetIncludes>...]]... Lists available rule sets and rules --exclude-rule[=<ruleExcludes>...] ID's of rules to exclude --exclude-rule-set[=<ruleSetExcludes>...] ID's of rule sets to exclude -h, --help Show this help message and exit. --include-rule[=<ruleIncludes>...] ID's of rules to include --include-rule-set[=<ruleSetIncludes>...] ID's of rule sets to include -o, --output=<output> Output file --progress-console Output progress to console --progress-data Output progress data --progress-json Output progress in JSON --progress-output=<progressOutput> Output file for progress monitor -r, --[no-]rules Output rules -V, --version Print version information and exit.
