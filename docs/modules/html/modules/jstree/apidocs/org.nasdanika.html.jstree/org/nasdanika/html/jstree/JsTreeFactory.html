      • buildAjaxJsTree

        String buildAjaxJsTree​(String nodesUrl,
                       String contextMenuUrl)
        Builds JsTree object for loading nodes and context menus using AJAX from the specified URL's. Node id is passed to the server side in "id" request parameter.
        Parameters:
        nodesUrl - URL to load nodes.
        contextMenuUrl - URL to load context menus. Can be null.
        Returns:
        String representation because JSON does not support functions.

        Generates script tag for binding jsTree to html element.
        Parameters:
        selector -
        jsTree -
        filter - Filter script to manipulate jsTree object before passing it to jsTree. It can be used to add functions to jsTree, which are not supported by JSON. The filter script can access jsTree via tree variable.
