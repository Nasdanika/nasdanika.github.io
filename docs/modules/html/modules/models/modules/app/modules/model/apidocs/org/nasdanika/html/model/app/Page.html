org.nasdanika.html.model.app

Interface Page

  • All Superinterfaces:
    Adaptable, BootstrapElement, EObject, HtmlElement, Marked, ModelElement, Notifier
    All Known Implementing Classes:
    PageImpl 

    public interface Page
extends BootstrapElement
    A representation of the model object 'Page'. Application page consisting of several parts - header, navigation bar, navigation panel, content panel, and footer. Page extends Tag, but ``name`` attribute shall not be used - it defaults to ``div``.

    The following features are supported:

    • Fluid
    • Header
    • Navigation Bar
    • Navigation Panel
    • Content Panel
    • Footer
    • Content Row Appearance
    See Also:
    AppPackage.getPage()
