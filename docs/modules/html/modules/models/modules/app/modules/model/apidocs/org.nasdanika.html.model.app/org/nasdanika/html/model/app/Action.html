Module org.nasdanika.html.model.app
Package org.nasdanika.html.model.app

Interface Action

  • All Superinterfaces:
    Adaptable, BootstrapElement, EObject, HtmlElement, Item, Label, Link, Marked, Marked, ModelElement, Notifier
    All Known Implementing Classes:
    ActionImpl 
    public interface Action
extends Link
    A representation of the model object 'Action'. Actions form a hierarchy. Application pages are generated from actions.

    The following features are supported:

    • Section Columns
    • Section Style
    • Sections
    • Navigation
    • Left Navigation
    • Right Navigation
    • Float Left Navigation
    • Float Right Navigation
    • Anonymous
    • Resources
    • Inline
    • Modal Activator
    See Also:
    AppPackage.getAction()
