org.nasdanika.html.model.app

Interface Label

  • All Superinterfaces:
    Adaptable, BootstrapElement, EObject, HtmlElement, Item, Marked, ModelElement, Notifier
    All Known Subinterfaces:
    Action, Link
    All Known Implementing Classes:
    ActionImpl, LabelImpl, LinkImpl 

    public interface Label
extends BootstrapElement, Item
    A representation of the model object 'Label'. Label is a text and an icon with a tooltip, notification badge, and help dialog. Labels can have children. Label is a base class for Link.

    The following features are supported:

    • Id
    • Text
    • Icon
    • Tooltip
    • Outline
    • Notification
    • Children
    • Help
    See Also:
    AppPackage.getLabel()
