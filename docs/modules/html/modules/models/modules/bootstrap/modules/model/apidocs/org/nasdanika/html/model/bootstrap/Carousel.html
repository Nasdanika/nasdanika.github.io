Interface Carousel

  • All Superinterfaces:
    Adaptable, BootstrapElement, Div, EObject, HtmlElement, Marked, ModelElement, Notifier, Tag
    All Known Implementing Classes:
    CarouselImpl 

    public interface Carousel
extends Div
    A representation of the model object 'Carousel'.

    The following features are supported:

    • Slides
    • Controls
    • Indicator
    • Cross Fade
    • Ride
    • Interval
