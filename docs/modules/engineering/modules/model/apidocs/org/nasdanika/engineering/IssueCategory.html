Interface IssueCategory

  • All Superinterfaces:
    Adaptable, Aim, Alignable, EObject, Marked, ModelElement, NamedElement, Notifier
    All Known Implementing Classes:
    IssueCategoryImpl 

    public interface IssueCategory
extends Aim
    A representation of the model object 'Issue Category'.

    The following features are supported:

    • Weight
    • Issues
    • Cumulative
    • Allocations
    • Children

      • isCumulative

        boolean isCumulative()
        Returns the value of the 'Cumulative' attribute. If true, unused category weight "capacity" is carried over to future increments and overused is subtracted from future increments. This allow low-weight categories to accumulate weight over time.
        Returns:
        the value of the 'Cumulative' attribute.
