    public interface Objective

    A representation of the model object 'Objective'.

    The following features are supported:

    • Endeavor
    • Key Results
    • Children
    • Parent
    • Linked Objectives
    • Sub Objectives

      • getEndeavor

        Endeavor getEndeavor()
        Returns the value of the 'Endeavor' reference. It is bidirectional and its opposite is 'Linked Objectives'. Endeavor for which containing [engineer](Engineer.html) has this objective for. E.g. "Achieve XYZ in Q2". Objectives can also be contained in endeavors directly. In this case they are assumed to be objectives of the assignee.
        Returns:
        the value of the 'Endeavor' reference.
        See Also:
        setEndeavor(Endeavor), EngineeringPackage.getObjective_Endeavor(), Endeavor.getLinkedObjectives()
