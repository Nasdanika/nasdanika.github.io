      • getGoals

        EList<Goal> getGoals()
        Returns the value of the 'Goals' containment reference list. The list contents are of type Goal. Persona goals are used to elicit and align [endeavors](Endeavor.html). E.g. new [product](Product.html) [features](Feature.html).
        Returns:
        the value of the 'Goals' containment reference list.
        See Also:
        EngineeringPackage.getPersona_Goals()

      • getRepresentatives

        EList<Engineer> getRepresentatives()
        Returns the value of the 'Representatives' reference list. The list contents are of type Engineer. It is bidirectional and its opposite is 'Represents'. Engineers which represent this persona. E.g. in a hierarchical [organization](Organization.html) some units or engineers may build products for internal customers - other units or engineers. In this case some of those internal customers act as persona representatives and work with the persona owner to identify persona goals and product features to address those goals.
        Returns:
        the value of the 'Representatives' reference list.
        See Also:
        EngineeringPackage.getPersona_Representatives(), Engineer.getRepresents()

      • getExtends

        EList<Persona> getExtends()
        Returns the value of the 'Extends' reference list. The list contents are of type Persona. A persona may extend another persona. E.g. "Moderator" discussion forum persona extends "User" persona, which may in turn extend "Visitor" persona. This persona inherits goals for the extended personas.
        Returns:
        the value of the 'Extends' reference list.
        See Also:
        EngineeringPackage.getPersona_Extends()
