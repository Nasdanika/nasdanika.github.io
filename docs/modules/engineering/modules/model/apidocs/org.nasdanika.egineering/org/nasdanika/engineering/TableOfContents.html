      • getRole

        String getRole()
        Returns the value of the 'Role' attribute. Table of contents role defines where it is positioned on the page. See [Appearance](Appearance.html) for details regarding page parts. One of the following: * ``content`` - in the content panel, before the document/section text. * ``content-left`` - Left content panel. * ``content-right`` - Right content panel. * ``float-left`` - Floats to the left. Diferent from ``content-left`` which takes the entire column. * ``float-right`` - Floats to the right. Different from ``content-right`` which takes the entire column.
        Returns:
        the value of the 'Role' attribute.
