Interface Marker

    All Known Subinterfaces:

    All Known Implementing Classes:
    GitMarkerImpl 

    public interface Marker
extends EObject, Marker
    A representation of the model object 'Marker'. Provides information about location for a model element definition - URI, line and column numbers.

    The following features are supported:

    • Location
    • Line
    • Column
    • Comment
    See Also:
    NcorePackage.getMarker()
