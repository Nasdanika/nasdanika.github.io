Interface NcorePackage.Literals

  • Enclosing interface:
    NcorePackage 

    public static interface NcorePackage.Literals
    Defines literals for the meta objects that represent
    • each class,
    • each feature of each class,
    • each operation of each class,
    • each enum,
    • and each data type
