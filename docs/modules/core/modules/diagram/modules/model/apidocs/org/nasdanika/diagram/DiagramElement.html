org.nasdanika.diagram

Interface DiagramElement

  • All Superinterfaces:
    EObject, Link, Notifier, Style
    All Known Subinterfaces:
    End, Start
    All Known Implementing Classes:
    DiagramElementImpl, EndImpl, StartImpl 

    public interface DiagramElement
extends Link, Style
    A representation of the model object 'Element'.

    The following features are supported:

    • Id
    • Name
    • Elements
    • Connections
    • Stereotype
    • Gradient
    • Border
    See Also:
    DiagramPackage.getDiagramElement()
