    public interface Style
extends EObject
    A representation of the model object 'Style'.

    The following features are supported:

    • Type
    • Notes
    • Description
    • Color
    • Dashed
    • Dotted
    • Bold
    • Properties

      • getType

        String getType()
        Returns the value of the 'Type' attribute. The default value is "". For connections - connection type. E.g. ``-->`` or ``..>``. To support styling connection type shall contain ``${style}`` token which will be expanded during generation. E.g. ``-${style}->``. For diagram elements - element type e.g. component, join, ...
        Returns:
        the value of the 'Type' attribute.
        See Also:
        setType(String), DiagramPackage.getStyle_Type()
