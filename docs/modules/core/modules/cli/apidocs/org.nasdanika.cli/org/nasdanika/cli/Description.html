Module org.nasdanika.cli
Package org.nasdanika.cli

Annotation Type Description

  • @Retention(RUNTIME)
@Target(TYPE)
public @interface Description
    Specifies a classloader markdown resource with a detailed description of a command. The description is output to HTML documentation below the usage information. By default, if value attribute is not set, a resource with the same name as the annotated type and .md extension is used for description if it exists.
    Author:
    Pavel
