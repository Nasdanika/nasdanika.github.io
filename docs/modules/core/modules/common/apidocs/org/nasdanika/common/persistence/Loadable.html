Interface Loadable

  • All Known Implementing Classes:
    FeatureObject, SupplierFactoryFeatureObject 

    public interface Loadable
    Objects which know how to load themselves from configuration may implement this interface. Also this interface may be used as an adapter interface. In this case an object to be loaded is first created by the object loader based on type passed to ObjectLoader's create() method and then is adapted to this interface to be loaded.
