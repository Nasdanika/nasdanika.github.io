Class PrintStreamProgressMonitor

  • java.lang.Object
    • org.nasdanika.common.PrintStreamProgressMonitor
  • All Implemented Interfaces:
    AutoCloseable, Composeable<ProgressMonitor>, ProgressMonitor 
    public class PrintStreamProgressMonitor
extends Object
implements ProgressMonitor
    Progress monitor reporting to a PrintStream, e.g. ``System.out``. This monitor indents sub-tasks and worked messages. It can be thought of as a hierarchical logger.
