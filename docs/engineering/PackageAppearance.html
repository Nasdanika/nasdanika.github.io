PackageAppearance

Package appearance contains maps of:

  • Package class names in kebab case to model element appearance
  • Sub-package names in kebab case to package apperance.

Package appearance allows to customize UI generation.

See default-appearance.yml for the default appearance definition, appearance.yml and getAppearanceLocations() in TestModel for an example of customization.
